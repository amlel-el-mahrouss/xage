NOTICE, 'XPX ADVANCED GAME ENGINE'
==================================

This is a template repository, containing code to make a basic game powered by XAGE.

REQUIREMENTS:
=============

- Git
- Mono
- GLFW
- OpenSSL
- LuaNgine
- FreeType
- OpenAL
- Direct3D11 debug layers.
- Windows 11, an XBOX
- Visual Studio 2022 (C++ Workload/UWP for C++)
- Vulkan SDK 1.3.250.1 (Vulkan Rendering System)
- DirectXTK for VS2019/Windows 10 (Install for Xplicit and XPXPlayer)

INSTALLING:
===========

- Put the C# assemblies into %appdata%/NginData
- Download DirectXTK for the SoundDriver.h header. (XPXPlayer) via NuGet.
